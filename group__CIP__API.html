S_CIP_Instance* addCIPInstances ( S_CIP_Class *  pa_pstCIPObject,
int  pa_nNr_of_Instances 
)

Add a number of CIP instances to a given CIP class.

The required number of instances are created in a block, but are attached to the class as a linked list. the instances are numbered sequentially – i.e. the first node in the chain is instance 1, the second is 2, etc. you can add new instances at any time (you do not have to create all the instances of a class at the same time) deleting instances once they have been created is not supported out-of-order instance numbers are not supported running out of memory while creating new instances causes an assert

Parameters
pa_pstCIPObjectCIP object the instances should be added
pa_nNr_of_Instancesnumber of instances to be generated.
Returns
pointer to the first of the new instances 0 on error
