Typedef Documentation

typedef struct CIP_MR_Object S_CIP_MR_Object

A class registry list node a linked list of this object is the registry of classes known to the message router for small devices with very limited memory it could make sense to change this list into an array with a given max size for removing the need for having to dynamically allocate memory. The size of the array could be a parameter in the platform config file.
