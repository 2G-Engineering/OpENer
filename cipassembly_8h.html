notify an Assembly object that data has been received for it.

The data will be copied into the assembly objects attribute 3 and the application will be informed with the IApp_after_assembly_data_received function.

Parameters
pa_pstInstancethe assembly object instance for which the data was received
pa_pnDatapointer to the data received
pa_nDatalengthnumber of bytes received
Returns

  • EIP_ERROR the received data was wrong
